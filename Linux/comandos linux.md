# Comandos de linux

### ls

List all files, but sorted by size and print the size.

flags:
* s size
* S sort

```
ls -sS
```
### cd

Move into the previous directory

```
cd -
```

### pwd

Print the current working directory

```
pwd
```



### echo

Print a value to the stardard output

```
echo "Hi Mom!"
```

```

```
